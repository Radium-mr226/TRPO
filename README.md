# TRPO - Технология Разработки Программного Обеспечения

Этот репозиторий содержит лабораторные и практические работы по дисциплине "Технология разработки программного обеспечения", выполненные в рамках обучения по специальности 09.02.07 "Информационные системы и программирование".

## Структура репозитория

* ЛабР_1: Лабораторная работа №1 - Разработка технического задания на примере приложения "Калькулятор".
* ЛабР_2: Лабораторная работа №2 - Работа с массивами (Пример: подсчет отрицательных элементов на диагонали).
* ЛабР_3: Лабораторная работа №3 - Составление диаграммы сущность–связь ERD.
* ЛабР_4: Практическая работа №4 - Разработка структурной и функциональной схемы приложения "Калькулятор".
* Контрольная_работа_1: Контрольная работа №1 - Разработка схем базы данных и архитектуры клиент-сервер.
* ЛабР_5: Лабораторная работа №5 - Диаграммы вариантов использования (прецедентов).
* ЛабР_6: Лабораторная работа №6 - Диаграммы последовательностей состояний и диаграммы деятельностей.
* ЛабР_7: Лабораторная работа №7 - Диаграммы классов.
* ЛабР_8: Лабораторная работа №8 - Разработка тестового сценария. Тестовые пакеты.
